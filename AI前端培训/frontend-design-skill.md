# Frontend Design Skill (前端设计高级技能) 使用指南

## 🌟 核心理念与目标

这个 `frontend-design` Skill 的核心宗旨是**彻底摆脱千篇一律的、死板的“AI味设计”（Generic AI Slop）**。它强制要求我们在生成 UI 时，采用具有独特审美、高级排版、和定制化交互的“专家级/高定级别”设计。

在项目中装备此技能后，我们在书写组件时将不再局限于泛泛而谈的“帮我写个卡片”，而是需要**明确地向 AI 指定极端的、带有强烈性格色彩的审美偏好 (Aesthetic Point-of-View)**。

---

## 💡 使用法则：给 AI 设定“极端的”风格基调

不要向 AI 提供平庸的需求，而是为其设定清晰的艺术指导方向。您可以直接套用以下几种常用指令模板来下达需求：

### 场景 1：要求极简与奢侈感 (Luxury / Refined)

> “请按照 `frontend-design` skill 的标准重构组件。采用**极简奢侈风 (luxury/refined)**。要求使用极其克制的留白（Controlled density）、高级的非系统默认字体（摒弃 Arial/Inter），配合细致的边框和非常优雅平滑的 CSS 微交互。不要使用任何毫无意义的渐变色。”

### 场景 2：要求打破常规的先锋设计 (Editorial / Brutalist)

> “采用**杂志排版风 (Editorial/Magazine)** 开启 frontend-design 技能。我希望看到不对称的布局 (Asymmetry)、重叠的元素 (Overlap) 和打破常规网格的构图。使用大胆的巨大衬线字体作为全屏背景，颜色采用低饱和度对比色。拒绝普通的左右对齐的版式！”

### 场景 3：要求硬壳科技感 (Industrial / Utilitarian)

> “使用 frontend-design 规范，采用**工业机能风 (Industrial/Utilitarian)** 重新设计这段布局。设计必须是硬朗的、纯 CSS 几何分割的，加入噪点背景质感 (noise textures) 和单色系的强对比。大量使用等宽字体 (Monospace) 凸显技术极客感。”

---

## 🛠️ 遵循的核心设计准则

当被调用时，我们会强制遵守以下前端美学（Frontend Aesthetics Guidelines）：

1. **绝对排斥陈词滥调**：向俗气的“白色背景+紫色模糊光晕渐变”等烂大街 AI 设计说不。坚决不使用千篇一律的组件库套路。
2. **排版为王 (Typography)**：挑选并引入非常具有辨识度的展示字体（例如特色 Serif 字体、Grotesque 无衬线体等），并配合精致的正文排版。
3. **空间构图 (Spatial Composition)**：
   - 拒绝大面积的无意义空无间距。
   - 使用**无负空间**或者**极密集展示 (Controlled density)** 来营造如同高级海报一般的呼吸感。
   - 大胆尝试对角线构图、非对称、以及打破传统网格的重叠元素。
4. **色彩与主题 (Color & Theme)**：使用主导色配以锐利刺目的强调色，避免温吞平均的调色盘。大胆使用暗黑风格或高饱和质感，引入网格渐变、噪点叠加等纹理元素来增加界面的厚度。
5. **精美动效 (Motion)**：在 HTML/React 中使用 `animation-delay` 的交错入场动画，或者制作令人惊艳的 CSS 悬停效果，甚至利用 Scroll-Trigger 给用户带来惊奇的操作反馈（Surprise）。

---

## 🚀 实践与验收标准

最后请谨记：**实现的代码复杂度必须与艺术构想相匹配！**

- 如果选择了极繁主义（Maximalist），代码就应该包含繁复的特效和多层视差动画；
- 如果选择了极简高级（Minimalist），那么必须展现出对像素级的控制、丝滑的留白、和每一个微小细节处理的绝对自律。

尽情发散思维，每次设计都要追求风格各异，展现真正的创意！
