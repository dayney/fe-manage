# AI工具安装指南 - 快速上手

## 🥇 强烈推荐：Cursor

### 为什么选择Cursor？
- ✅ 专为开发者设计，AI能力最强
- ✅ 与VSCode类似，学习成本低
- ✅ 代码理解能力强
- ✅ 支持多文件上下文
- ✅ 免费版功能已足够强大

### 安装步骤

1. **下载安装**
   - 访问：https://cursor.sh
   - 下载对应系统版本
   - 安装（与VSCode安装类似）

2. **基本设置**
   - 打开Cursor
   - 登录账号（支持GitHub/Google登录）
   - 免费版每天有使用限额，但足够日常使用

3. **快速上手**
   - 打开项目文件夹
   - `Cmd/Ctrl + K`：AI编辑代码
   - `Cmd/Ctrl + L`：AI聊天
   - `@文件名`：引用文件到上下文
   - `@函数名`：引用函数到上下文

4. **使用技巧**
   ```
   - 选中代码 + Cmd/Ctrl + K：让AI修改代码
   - 选中代码 + Cmd/Ctrl + L：询问AI代码问题
   - 在聊天中使用@符号引用文件/函数
   - 右键代码 → "Explain"：解释代码
   - 右键代码 → "Refactor"：重构代码
   ```

### 费用
- 免费版：每天有使用限额
- Pro版：$20/月（团队使用建议）

## 🥈 GitHub Copilot

### 为什么选择Copilot？
- ✅ VSCode插件，无缝集成
- ✅ 代码补全能力强
- ✅ 支持多种语言
- ✅ 微软官方支持

### 安装步骤

1. **安装插件**
   - 打开VSCode
   - 搜索"GitHub Copilot"
   - 点击安装

2. **激活账号**
   - 需要GitHub账号
   - 学生/开源项目可申请免费
   - 个人版：$10/月

3. **使用方式**
   - 输入注释，AI自动补全代码
   - `Tab`：接受建议
   - `Alt + ]`：下一个建议
   - `Alt + [`：上一个建议

### 费用
- 学生：免费
- 个人：$10/月
- 企业：$19/用户/月

   - 免费版可用

## ❓ 常见问题

### Q: 哪个工具最好？
**A:** 对于前端开发，**Cursor**是最佳选择，功能全面且易用。

### Q: 免费版够用吗？
**A:** 对于个人学习和日常使用，免费版通常足够。团队使用建议付费版。

### Q: 需要同时安装多个工具吗？
**A:** 建议从Cursor开始，熟练后再根据需要添加其他工具。

### Q: 数据安全吗？
**A:** 
- Cursor：代码不会上传到训练模型
- GitHub Copilot：遵循GitHub隐私政策
- ChatGPT：注意不要上传敏感代码

### Q: 如何选择付费版？
**A:** 
- 个人使用：Cursor免费版 + ChatGPT免费版
- 团队使用：Cursor Pro团队版
- 预算有限：Codeium（完全免费）
